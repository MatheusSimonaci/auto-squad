## Angle Diagnostic

### Selected Angle
**Type:** Provocative + Self-Recognition
**Rationale:** Aplicando Ângulo Contraintuitivo — a caption questiona o que a audiência acredita saber ("incluindo, talvez, você"). Para cold traffic, o ângulo de "você está fazendo errado" é o de maior CTR porque ativa a autoconsciência sem ser acusatório. O "talvez" suaviza o desafio e mantém o tom acolhedor do canal.

Aplicando Parada de Rolagem: os primeiros 5 palavras — "A maioria das pessoas cita" — criam identificação social imediata (todo mundo faz isso → logo eu provavelmente também). O leitor para o scroll para descobrir se está incluído no "talvez".

### Alternatives Considered

| Angle | Draft | Rejection Reason |
|-------|-------|-----------------|
| Curiosity Tease (topic) | "O que a Bíblia realmente diz sobre dinheiro 📖" | Fraco para cold traffic — não distingue este vídeo de outros 1000 sobre o mesmo tema. Sem curiosity gap, sem parada de scroll. |
| Emotional Resonance | "Esse versículo me liberou de uma culpa que carregava há anos 📖" | Funciona para warm/hot audience (seguidores que confiam no canal). Para FYP frio, parece anúncio de produto pessoal. Descartado para esta distribuição. |
| Direct Value | "1 Tm 6:10 — o que o texto realmente diz sobre dinheiro" | Muito seco para TikTok. Funciona no YouTube mas não tem tensão emocional para parar scroll em feed. |

## Temperature Diagnosis

**Assumed Temperature:** Cold (FYP/discovery)

Este vídeo será distribuído primariamente pelo algoritmo de descoberta do TikTok para cristãos brasileiros que ainda não conhecem o canal. A maioria dos espectadores chegará via FYP, não por seguir o canal. Portanto, a caption não pode assumir contexto (ex: "como mencionei no vídeo anterior") e precisa funcionar como primeira impressão standalone.

Audiência cold sobre este tema tem crença prévia forte: a maioria carrega a versão errada ("dinheiro é raiz de todo mal") como verdade bíblica. O ângulo contraintuitivo funciona especificamente porque desafia uma crença consolidada de forma gentil — "talvez você" evita acusação direta.

Ajuste de copy para temperature: caption não fecha o loop (não resolve a dúvida) — o vídeo faz isso. A caption deixa a tensão aberta: "incluindo, talvez, você" é um open loop que o espectador fecha assistindo.
