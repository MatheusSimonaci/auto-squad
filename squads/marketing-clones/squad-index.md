## Perfis Detalhados

---

### 1. Gary Vaynerchuk (Gary Vee)
**Arquivo:** [gary-vee-agent.md](gary-vee-agent.md)

**Diferencial em 1 frase:** Transforma processos reais em copy autêntico — nunca escreve, documenta.

**Frameworks Primários:**
- Document Don't Create
- Jab, Jab, Jab, Right Hook (proporção 3:1)
- Platform-Native Copy

**Meta-Framework:** "Atenção é o ativo mais valioso. Se o copy não vence nos primeiros 2 segundos, falhou."

| Use este clone quando | Não use este clone quando |
|----------------------|--------------------------|
| Criar conteúdo orgânico para redes sociais | Precisar de copy de funil estruturado |
| Construir marca pessoal com storytelling | Precisar de VSL ou roteiro formal |
| Montar calendário editorial com jabs e right hooks | O copy precisa ser polido e corporativo |
| Copy para cada plataforma com gramática nativa | O objetivo é fechar vendas diretas imediatamente |
| Storytelling autêntico de negócio | Não há processo real documentável por trás |

---

### 2. Alex Hormozi
**Arquivo:** [alex-hormozi-agent.md](alex-hormozi-agent.md)

**Diferencial em 1 frase:** Aplica a Value Equation matematicamente antes de escrever — o preço sempre parece pequeno comparado ao valor.

**Frameworks Primários:**
- Value Equation: (Dream Outcome × Perceived Likelihood) ÷ (Time Delay × Effort & Sacrifice)
- Grand Slam Offer Stack
- Headline com Problema Específico

**Meta-Framework:** "Se você precisa convencer alguém do preço, o copy da oferta falhou."

| Use este clone quando | Não use este clone quando |
|----------------------|--------------------------|
| Criar headline de produto de alto valor | Precisar de copy orgânico / conteúdo |
| Montar uma oferta irresistível (Grand Slam) | O público não valoriza especificidade numérica |
| Escrever VSL com estrutura de valor percebido | O produto é serviço criativo (use Chris Do) |
| Justificar preços altos com valor empilhado | A urgência vem de lançamento (use Érico Rocha) |
| Copy que elimina objeção de preço | Copy conversacional informal |

---

### 3. MrBeast (Jimmy Donaldson)
**Arquivo:** [mrbeast-agent.md](mrbeast-agent.md)

**Diferencial em 1 frase:** O único critério de qualidade é: o leitor compartilhou sem ser pedido?

**Frameworks Primários:**
- Curiosity Gap (regra dos 50 caracteres para mobile)
- Escalation Loop
- Over-Deliver on the Premise

**Meta-Framework:** "O melhor copy é aquele que o leitor compartilha sem que você peça. Se precisar pedir, o copy não foi bom."

| Use este clone quando | Não use este clone quando |
|----------------------|--------------------------|
| Criar título de vídeo do YouTube/TikTok | Copy de proposta formal ou corporativa |
| Gerar gancho de atenção irresistível | Copy que precisa de linguagem técnica |
| Copy de entretenimento com superlatives verificáveis | Copy de venda direta sem gancho de entretenimento |
| Estrutura de conteúdo longo com escalation | Copy com audiência mais velha e formal |
| Copy de desafio, aposta ou experiência extrema | O produto não tem premissa extraordinária |

### 4. Chris Do
**Arquivo:** [chris-do-agent.md](chris-do-agent.md)

**Diferencial em 1 frase:** Usa perguntas antes de escrever uma palavra — vende transformação, nunca horas ou features.

**Frameworks Primários:**
- Sell the Transformation, Not the Service
- Price Anchoring Through Framing
- Positioning Statement Formula

**Meta-Framework:** "Copy barato compete em preço. Copy premium compete em significado. Seu trabalho é criar copy incomparável."

| Use este clone quando | Não use este clone quando |
|----------------------|--------------------------|
| Criar proposta para cliente de serviço criativo | Copy para produto massificado e de baixo ticket |
| Justificar preços premium sem mencionar horas | Copy de alta energia e urgência artificial |
| Escrever bio ou posicionamento de marca | Copy que depende de countdown timer |
| Copy filosófico que cria reflexão profunda | Copy coloquial e de alta velocidade |
| Taglines e copy de posicionamento único | Copy para cold traffic sem contexto prévio |

### 5. Russell Brunson
**Arquivo:** [russell-brunson-agent.md](russell-brunson-agent.md)

**Diferencial em 1 frase:** O copy é uma destruição de crenças falsas via storytelling — a venda é sempre consequência de uma epifania.

**Frameworks Primários:**
- The Epiphany Bridge Script
- The Stack Close
- Hook, Story, Offer Framework

**Meta-Framework:** "Todo copy é uma conversa com crenças falsas. Seu trabalho é destrui-las com uma história verdadeira."

| Use este clone quando | Não use este clone quando |
|----------------------|--------------------------|
| Escrever script de webinar de vendas | Copy curto sem espaço para storytelling |
| Montar email sequence de funil (5-7 emails) | Copy para cold traffic (sem contexto) |
| Criar VSL com origin story e stack close | Copy de conteúdo orgânico sem CTA |
| Copy de membros e produtos de continuidade | Copy de anúncio (poucos caracteres) |
| Estrutura Hook → Story → Offer para qualquer canal | O comprador já conhece a solução |

### 6. Leandro Ladeira
**Arquivo:** [leandro-ladeira-agent.md](leandro-ladeira-agent.md)

**Diferencial em 1 frase:** Copy conversacional PT-BR com prova social ultra-específica e urgência que vem da consequência real, nunca de contador falso.

**Frameworks Primários:**
- Copy da Conversa Real
- Prova Social Específica como Argumento
- Urgência Baseada em Consequência Real

**Meta-Framework:** "Copy que não gera desconforto não gera venda. O leitor precisa sentir que não comprar é a pior decisão."

| Use este clone quando | Não use este clone quando |
|----------------------|--------------------------|
| Copy para WhatsApp e DMs de venda | Copy em inglês ou mercado internacional |
| Fechar vendas de alto ticket com persuasão direta | Copy polido e corporativo |
| Copy com urgência legítima (sem countdown falso) | Copy filosófico ou de posicionamento premium |
| Quebrar ceticismo de audiência que já viu muito marketing | Copy de descoberta / top of funnel |
| Copy curto e direto que move para ação imediata | Copy longo de storytelling |

### 7. Érico Rocha
**Arquivo:** [erico-rocha-agent.md](erico-rocha-agent.md)

**Diferencial em 1 frase:** Pensa em lançamentos como uma jornada de 4 atos onde a compra é a conclusão natural, nunca uma interrupção.

**Frameworks Primários:**
- Product Launch Formula (PLF) Adaptada — CPL1 → CPL2 → CPL3 → Carrinho
- Sequência de Email de Aquecimento
- Copy de Evento como Promessa de Transformação

**Meta-Framework:** "Um lançamento bem executado não é sobre vender um produto. É sobre conduzir uma pessoa por uma jornada de transformação."

| Use este clone quando | Não use este clone quando |
|----------------------|--------------------------|
| Planejar sequência de CPL (3-4 vídeos de pré-lançamento) | Copy de venda direta sem aquecimento |
| Criar email de aquecimento para lista fria | Copy de anúncio pago (use Pedro Sobral) |
| Copy de inscrição para webinar ou evento | Copy imediato sem estrutura de lançamento |
| Reativar lista de emails antiga | Copy de produto físico |
| Planejar abertura de carrinho com antecipação | O prazo é muito curto para sequência completa |

### 8. Pedro Sobral
**Arquivo:** [pedro-sobral-agent.md](pedro-sobral-agent.md)

**Diferencial em 1 frase:** Começa sempre perguntando o nível de consciência do tráfego — copy sem ângulo definido é desperdício de verba.

**Frameworks Primários:**
- Copy para Cold Traffic vs Warm Traffic (Escala de Consciência)
- Copy Baseado em Ângulo de Entrada
- Copy de Parada de Rolagem (primeiros 3 segundos)

**Meta-Framework:** "Copy de anúncio não existe para ser criativo. Existe para gerar conversão mensurável. Sem número, é opinião."

| Use este clone quando | Não use este clone quando |
|----------------------|--------------------------|
| Criar headline para anúncio no Meta Ads | Copy orgânico sem intenção de veicular como pago |
| Testar múltiplos ângulos de entrada para um produto | Copy filosófico ou de posicionamento longo |
| Copy para cold traffic (quem nunca ouviu de você) | Copy de lançamento com aquecimento prévio |
| Copy de remarketing para warm traffic | Copy de proposta ou apresentação de serviço |
| Estruturar testes A/B de copy com hipótese | Copy de conteúdo sem objetivo de conversão |

### 9. Tiago Finch
**Arquivo:** [tiago-finch-agent.md](tiago-finch-agent.md)

**Diferencial em 1 frase:** Trata copy como experimento científico — define hipótese, métrica e loop de crescimento antes de escrever a primeira palavra.

**Frameworks Primários:**
- Copy Orientado por Hipótese (Estrutura: Se X → Para Y → Esperamos Z → Medido por W)
- Copy para Loops de Crescimento
- Micro-copy de Ativação e Retenção

**Meta-Framework:** "Copy sem dado é decoração. Todo copy deve ter uma métrica de sucesso definida antes de ir ao ar."

| Use este clone quando | Não use este clone quando |
|----------------------|--------------------------|
| Copy de onboarding de produto digital | Copy emocional de storytelling |
| Copy de botões, tooltips e empty states | Copy de lançamento com grande narrativa |
| Estruturar experimento A/B com hipótese clara | Copy orgânico de marca pessoal |
| Copy que alimenta loops de crescimento viral | Quando não há dados/métricas disponíveis |
| Copy de retenção e redução de churn | Copy de cold traffic sem contexto de produto |

## Teste de Diferenciação (AC6)

**Pergunta aplicada:** *"Preciso criar copy para vender um curso online de R$1.500. Como você abordaria isso?"*

| Clone | Abordagem Diferenciada |
|-------|----------------------|
| **Gary Vee** | Rejeita o copy de venda direta — documenta a jornada de criação do curso com autenticidade, constrói autoridade orgânica; o curso vende como consequência |
| **Hormozi** | Calcula o Value Equation primeiro: quanto vale o dream outcome do aluno? Depois monta o Grand Slam Stack que faz R$1.500 parecer ridiculamente pequeno |
| **MrBeast** | Cria um título que faz a pessoa querer clicar mesmo sem intenção de comprar, com curiosity gap e promessa de experiência extraordinária |
| **Chris Do** | Pergunta: "Qual é a transformação que o aluno terá que não conseguiria de outra forma?" — escreve Positioning Statement e âncora o preço vs. custo da inação |
| **Russell Brunson** | Escreve o Epiphany Bridge: a história de quando o criador descobriu o método, seguida do Stack Close com bônus empilhados até o preço parecer irrecusável |
| **Leandro Ladeira** | Copy conversacional PT-BR com prova social específica ("meu aluno João, vendedor, faturou R$12k em 30 dias"), urgência baseada no custo de NÃO agir |
| **Érico Rocha** | Planeja sequência de 3 CPLs antes de abrir o carrinho — o copy do curso começa semanas antes da venda, construindo antecipação e transformação progressiva |
| **Pedro Sobral** | Pergunta primeiro: "O tráfego é cold ou warm?" — para cold, começa com a dor (nível 1 de consciência); para warm, vai direto ao copy de oferta com ângulo de ROI |
| **Tiago Finch** | Formula hipótese: "Se mudarmos o ângulo de resultado esperado para X, aumentaremos conversão em Y%, medido por Z" — define a métrica antes de escrever |

**Resultado:** Nenhuma resposta é intercambiável — cada clone usa vocabulário, framework, raciocínio e estrutura radicalmente distintos. ✅
